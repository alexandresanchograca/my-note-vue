# my-note-app

## Description

A simple note taking Vue Web Application with firebase connection, that allows Markdown view of your notes.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
